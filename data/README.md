# データ（有休データのバックアップ置き場）

このフォルダは、アプリに入力した有休データ（JSON エクスポート）を保存して
おくための場所です。アプリ本体（`index.html`）とは別に、ここに
「データそのもの」を置いてバックアップします。

> ⚠️ **重要：有休データは個人情報です。**
> このリポジトリは公開（ポートフォリオ）を想定しているため、
> `data/*.json` は `.gitignore` で **git 管理の対象外** にしています。
> JSON ファイルはコミット・公開されません。手元のPC内にのみ残ります。

## バックアップの手順

1. ブラウザで `index.html` を開く
2. 「設定」タブ →「JSON エクスポート」をクリック
3. ダウンロードされた `selfman_leave_YYYY-MM-DD.json` を **このフォルダに移動**

`data/*.json` は git 管理外なので、PC が壊れた場合に備えるには
**このフォルダ自体を別の場所（クラウドストレージ・USB・別ドライブ等）にも
コピー**しておいてください。git では守られません。

## データを復元するとき

1. このフォルダから戻したい JSON ファイルを選ぶ
2. アプリの「設定」タブ →「インポート」でその JSON を読み込む

## ファイル名について

エクスポートされるファイル名は `selfman_leave_<日付>.json` です。
日付が入るので、上書きされず世代ごとに残せます。
最新版を分かりやすくしたい場合は `selfman_leave_latest.json` のように
別名で1つ置いておくのもおすすめです。
