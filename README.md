# 有休管理 - Self Manager

> 有給休暇の付与・取得・計画を一元管理する、フレームワーク不使用の単一ファイル Web アプリ。

`index.html` をブラウザで開くだけで動作します。サーバー・ビルド・インストールは一切不要で、
外部ライブラリ（CDN含む）にも依存しない、約1,000行の自己完結アプリです。

## デモ

**🔗 https://sam-okuda.github.io/self-man/**

上記の GitHub Pages 上でブラウザから直接お試しいただけます
（ローカルでは `index.html` を開くだけでも動作します）。

## 特徴

- **ゼロ依存・単一ファイル**: HTML / CSS / JavaScript のみ。フレームワーク・ビルドツール・外部CDN不使用
- **データはブラウザ内に保存**: `localStorage` を使用。入力情報は外部に送信されません
- **連休シミュレーション**: 今後90日以内に「有休1日で3連休以上」になる日を自動計算
- **可視化**: 取得・計画状況をグラフとカレンダーで表示（描画もライブラリなしで実装）
- **ダークモード対応**: ワンタップ切り替え、設定は保存
- **レスポンシブ**: PC・スマートフォン両対応
- **データ移行**: JSON / CSV エクスポート、JSON インポート対応

## 機能（タブ）

| タブ | 内容 |
| --- | --- |
| ダッシュボード | 付与日数・取得済み・計画中・残日数を一覧表示。連休シミュレーション付き |
| 取得履歴 | 取得済み有休の記録。追加・編集・削除、計画への差し戻しが可能 |
| 取得計画 | 今後の取得予定を登録。確定すると取得履歴へ移動。消化ペースの目安を表示 |
| カレンダー | 取得済み・計画中の日を月単位で表示 |
| グラフ | 取得・計画状況をグラフで可視化 |
| 設定 | 付与日数の設定、JSON/CSV エクスポート、インポート、データリセット |

## 使い方

1. `index.html` を Web ブラウザで開く
2. 各タブから有休の付与・取得・計画を入力する

データはお使いのブラウザの `localStorage` に保存されます。
別のブラウザ・端末・シークレットウィンドウでは引き継がれない点にご注意ください。

### データのバックアップ

データはブラウザ内にしかないため、定期的にバックアップを取ることを推奨します。
「設定」タブの「JSON エクスポート」で出力したファイルを `data/` フォルダに置きます。

なお、有休データは個人情報のため、`data/*.json` は `.gitignore` で git 管理対象外
としています（公開リポジトリには含まれません）。詳しい手順とバックアップ上の注意は
[`data/README.md`](data/README.md) を参照してください。

## 技術構成

- **言語**: HTML / CSS / JavaScript（バニラ、フレームワーク不使用）
- **データ保存**: ブラウザの `localStorage`
- **外部依存**: なし（CDN・npm パッケージ不使用）
- **グラフ・カレンダー**: ライブラリを使わず自前で描画
