# 有休管理 - Self Manager

有給休暇の付与・取得・計画を一元管理するための、単一ファイルの Web アプリです。
`index.html` をブラウザで開くだけで動作し、サーバーやインストールは不要です。

## 特徴

- **単一 HTML ファイル**: `index.html` だけで完結（外部依存・ビルド不要）
- **データはブラウザ内に保存**: `localStorage` を使用するため、入力した情報は外部に送信されません
- **ダークモード対応**: ヘッダーのトグルで切り替え（設定は保存されます）
- **レスポンシブ**: PC・スマートフォンの両方に対応

## 機能（タブ）

| タブ | 内容 |
| --- | --- |
| ダッシュボード | 付与日数・取得済み・計画中・残日数を一覧表示。今後90日以内に有休1日で3連休以上になる日を自動計算する連休シミュレーション付き |
| 取得履歴 | 取得済み有休の記録。追加・編集・削除、計画への差し戻しが可能 |
| 取得計画 | 今後の取得予定を登録。確定すると取得履歴へ移動。消化ペースの目安を表示 |
| カレンダー | 取得済み・計画中の日を月単位で表示 |
| グラフ | 取得・計画状況をグラフで可視化 |
| 設定 | 付与日数などの各種設定 |

## 使い方

1. `index.html` を Web ブラウザで開く
2. 各タブから有休の付与・取得・計画を入力する

データはお使いのブラウザの `localStorage` に保存されます。
別のブラウザ・端末・シークレットウィンドウでは引き継がれない点にご注意ください。

### データのバックアップ

データはブラウザ内にしかないため、定期的にバックアップを取ることを推奨します。
「設定」タブの「JSON エクスポート」で出力したファイルを `data/` フォルダに置きます。

なお、有休データは個人情報のため、`data/*.json` は `.gitignore` で git 管理対象外
としています（公開リポジトリには含まれません）。詳しい手順とバックアップ上の注意は
[`data/README.md`](data/README.md) を参照してください。

## 技術構成

- HTML / CSS / JavaScript（フレームワーク不使用）
- データ保存: ブラウザの `localStorage`
